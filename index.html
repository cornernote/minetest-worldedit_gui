WorldEdit GUI
for Minetest

Provides a player interface for worldedit mod.

Download GitHub Minetest CoRNeRNoTe

Features

  • Easily manipulate your Minetest world.
  • Supports all WorldEdit methods using a visual interface.

Docs

Requirements

  • Minetest >=0.4.3
  • Inventory Plus >=1.0.0

Installation

Download and uncompress the latest version, and move the the minetest-worldedit_gui-master folder into your minetest/mods folder.
